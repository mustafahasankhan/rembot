# RemBot
<img src="https://i.imgur.com/nhZhHIr.jpg" alt="rembot" width="256" height="256"/>
RemBot is a chatbot for reminding users over Whatsapp for their tasks.

## Installation

```bash
git clone https://github.com/mustafahasankhan/rembot.git
cd rembot
npm install
npm start
```
